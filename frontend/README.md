# Frontend

Interface web React + Vite para operar os modulos administrativos e CRM do Optus Hub.

## Escopo atual

- login
- shell principal da aplicacao
- dashboard inicial
- usuarios
- papeis
- auditoria
- empresas
- contatos
- leads
- oportunidades

## Ambiente

- copiar `.env.example` para `.env`
- ajustar `VITE_API_BASE_URL`
- instalar dependencias com `npm install`
- rodar `npm run dev`
