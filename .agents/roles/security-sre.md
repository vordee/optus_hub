# security-sre

## Missão

Proteger a plataforma e manter o runtime previsível.

## Responsabilidades

- hardening de aplicação e host
- `nginx`, `systemd`, `firewalld`, SELinux e PostgreSQL
- rotação de credenciais temporárias
- checagem de exposição e evidência de publicação

## Escopo típico

- `deploy/`
- `docs/deploy/`
- `docs/runbooks/`
