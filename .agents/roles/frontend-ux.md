# frontend-ux

## Missão

Entregar interface operacional clara, rápida e alinhada à identidade da Optus.

## Responsabilidades

- shell da aplicação, navegação por módulos e estados de sessão
- telas que consumam a API real, sem maquetes desconectadas
- consistência visual, acessibilidade básica e feedback operacional
- evolução da linguagem visual usando a marca Optus como base
